## 2. 配線図 (ピンアサイン)

Google Voice HAT互換の配線（I2Sクロック共有）を行っています。
**注意:** マイクは3.3V、アンプは5V推奨です。

| 物理ピン番号 | GPIO番号 | 役割 | 接続先 (マイク) | 接続先 (アンプ) | 接続先 (ボタン) |
| --- | --- | --- | --- | --- | --- |
| **1** | - | **3.3V** | **3V3 / VDD** | - | - |
| **2** | - | **5V** | - | **Vin** | - |
| **6** | - | **GND** | **GND** | **GND** | - |
| **12** | **GPIO 18** | **BCLK** | **SCK / BCLK** | **BCLK** | - |
| **14** | - | **GND** | - | - | **足②** |
| **16** | **GPIO 23** | **Button** | - | - | **足①** (対角推奨) |
| **35** | **GPIO 19** | **LRCLK** | **WS / LRCL** | **LRC** | - |
| **38** | **GPIO 20** | **Data In** | **SD / DOUT** | - | - |
| **40** | **GPIO 21** | **Data Out** | - | **DIN** | - |

* **マイク・アンプ共通:** BCLK (Pin 12) と LRCLK (Pin 35) は分岐して両方に接続。
* **マイク設定:** SELピンがある場合は GND に接続（左チャンネル/モノラル設定）。
* **アンプ設定:** GAIN, SDピンは未接続（デフォルト設定）。

## 3. システム設定 (OS)

### 3-1. `config.txt` の編集

音声ドライバとして `googlevoicehat-soundcard` を使用します。

コマンド: `sudo nano /boot/firmware/config.txt`
(古いOSの場合は `/boot/config.txt`)

**変更・追記内容:**

```ini
# 標準オーディオを無効化
#dtparam=audio=on

# Google Voice HATドライバを有効化 (ファイルの末尾に追記)
dtoverlay=googlevoicehat-soundcard

```

### 3-2. 必要なライブラリのインストール

Pythonで音声を扱うためのライブラリと、数値計算用ライブラリを入れます。

```bash
sudo apt-get update
sudo apt-get install -y python3-pyaudio python3-numpy python3-gpiozero python3-requests

```

※ `audioop` は最新Pythonで削除されたため、代わりに `numpy` を使用します。

### 3-3. 音量設定 (ALSA)

再生音量が小さい場合、OS側のマスターボリュームを上げます。

```bash
alsamixer

```

1. `F6` キーでサウンドカード (`googlevoicehat` / card 1) を選択。
2. `F3` (Playback) タブで音量を上げられるか確認 (ない場合はプログラム側で制御)。
3. 設定保存: `sudo alsactl store`

## 4. プログラム (Python)

**ファイル名:** `voice_echo_v4.py`

**機能:**

* ボタンを押している間だけ録音。
* **開始時のポップノイズ除去** (最初の数フレームを捨てる)。
* **終了時のクリックノイズ除去** (最後の数フレームをカット)。
* **ソフトウェア増幅** (NumPyを使用し、マイク入力を16倍〜に増幅)。
* **排他制御** (再生中は次の録音ができないようにブロック)。

```python
import pyaudio
import wave
from gpiozero import Button
import time
import subprocess
import numpy as np  # 音声増幅・加工用

# --- 設定パラメータ ---
BUTTON_PIN = 23              # ボタンのGPIOピン
WAVE_OUTPUT_FILENAME = "output.wav"
CHUNK = 4096                 # バッファサイズ
FORMAT = pyaudio.paInt32     # I2Sマイクは32bit転送
CHANNELS = 1                 # モノラル
RATE = 48000                 # サンプリングレート (48kHz)
DEV_INDEX = 1                # arecord -l で確認したカード番号

# ノイズ対策設定
START_TRIM_CHUNKS = 10       # 録音開始時に捨てるデータ数 (安定化待ち)
END_TRIM_CHUNKS = 5          # 録音終了時にカットするデータ数 (ボタン離音対策)

# 【音量調整】倍率 (マイク感度調整)
# 音が小さい場合は 16.0 -> 32.0 などに変更
VOLUME_GAIN = 16.0

# ボタン設定 (プルアップ有効)
button = Button(BUTTON_PIN, pull_up=True)

def play_audio(filename):
    """ 音声を再生する (aplayコマンドを使用) """
    print(f"🔊 再生中...")
    # -D plughw:1,0 でデバイス指定, -q でログ抑制
    subprocess.run(["aplay", "-D", "plughw:1,0", "-q", filename])
    print("✅ 再生終了")

def record_audio():
    """ マイクから録音し、増幅して保存する """
    audio = pyaudio.PyAudio()

    try:
        stream = audio.open(format=FORMAT,
                            channels=CHANNELS,
                            rate=RATE,
                            input=True,
                            input_device_index=DEV_INDEX,
                            frames_per_buffer=CHUNK)
    except Exception as e:
        print(f"エラー: マイクが開けません。 {e}")
        return False

    print("🔴 録音開始...（ボタンを離すと停止）")
    frames = []

    # 1. 開始ノイズ対策：最初の数回分を空読みして捨てる
    for _ in range(START_TRIM_CHUNKS):
        try:
            stream.read(CHUNK, exception_on_overflow=False)
        except:
            pass

    # 2. メイン録音ループ (ボタンが押されている間)
    while button.is_pressed:
        try:
            data = stream.read(CHUNK, exception_on_overflow=False)
            
            # --- ソフトウェア増幅処理 (Numpy) ---
            # バイナリデータをint32配列に変換
            signal = np.frombuffer(data, dtype=np.int32)
            
            # 音量を掛け算 (GAIN倍)
            signal = signal * VOLUME_GAIN
            
            # 音割れ防止 (クリッピング)
            signal = np.clip(signal, -2147483648, 2147483647)
            
            # 再びバイナリデータに戻す
            amplified_data = signal.astype(np.int32).tobytes()
            
            frames.append(amplified_data)
        except IOError:
            pass 

    print("⬛ 録音終了（処理中...）")

    # 3. 終了ノイズ対策：最後のお尻部分をカット
    if len(frames) > END_TRIM_CHUNKS:
        frames = frames[:-END_TRIM_CHUNKS]

    # ストリーム終了処理
    stream.stop_stream()
    stream.close()
    audio.terminate()

    # ファイル保存
    wf = wave.open(WAVE_OUTPUT_FILENAME, 'wb')
    wf.setnchannels(CHANNELS)
    wf.setsampwidth(audio.get_sample_size(FORMAT))
    wf.setframerate(RATE)
    wf.writeframes(b''.join(frames))
    wf.close()
    
    return True

def main():
    print("=== AIボイス端末システム (Echo Test) ===")
    print(f"現在の音量増幅率: {VOLUME_GAIN}倍")
    print("準備完了。ボタンを押して話しかけてください。")

    try:
        while True:
            # 1. ボタンが押されるのを待つ
            button.wait_for_press()
            
            # 2. 録音実行
            if record_audio():
                # 3. 録音が正常終了したら再生 (再生中は待機)
                play_audio(WAVE_OUTPUT_FILENAME)
            
            # 4. チャタリング防止の休憩
            time.sleep(0.5)

    except KeyboardInterrupt:
        print("\n終了します")

if __name__ == "__main__":
    main()

```
